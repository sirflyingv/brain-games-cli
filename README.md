### Installation

- clone repo

- run `make install`

- run `make publish`

- register package with `sudo npm link`

### Usage

Run in terminal one of the games by name: `brain-even`, `brain-calc`, `brain-gcd`, <br/> `brain-progression` or `brain-prime`.
